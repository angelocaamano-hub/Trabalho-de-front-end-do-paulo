CSS e Listas do que mais gosto

O "CSS" é importante para toda pagina web poque deixa a pagina mais interesante e com menos desordem visual e as listas são criadas para organizar todo o que voce vai fazer no trabalho ou no dia a dia.

TIPOS DE LISTAS

Aqui criamos 4 tipos de listas para que cada pessoa saiba o que eu gosto

  • Lista de dados pessoais
  • Lista de Jogos
  • Lista de filmes
  • Lista de hobbyes

A lista de dados pessoais é uma lista criada para falar os dados pessoais de uma pessoa para saber o que cada pessoa faz é essa lista facilita muito na hora de saber coisas de voce.

A lista de jogos seria uma lista feita para que os meus amigos possam saber o tipo de jogo que eu gosto.

A lista de filmes é a lista que informa aos meus amigos que quando eles quiserem ir a assistir algum filme eles tenham alguma ideia de qual filmes eu gosto.

A lista de hobbyes, a lista que mostra aos meus amigos as coisas que eu gosto e passo mais tempo.
